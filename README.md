# Space Shooter

A 2D arcade Space Shooter built using the **Godot Engine**. 

Play it directly in your browser: **[Play Space Shooter](https://emotionalape.github.io/space-shooter/)**

---

## Gameplay & Features

- **Player Spacecraft**: Navigate your ship in 2D space and fire lasers to destroy incoming obstacles.
- **Dynamic Starfield**: Features a starry background with randomized scales, positions, and speed offsets.
- **Randomized Obstacles (Meteors)**: Meteors spawn off-screen with randomized sizes, rotation rates, velocities, and textures (selected randomly).
- **Destructible Environment**: Shoot meteors with your ship's laser to destroy them, triggering explosion audio.
- **Score System**: Score is based on the elapsed time survived.
- **UI & Health Tracking**: Tracks and displays remaining player lives visually alongside the current score.
- **Game Over & Restart Flow**: Game Over screen showing your final score, restart by pressing the SPACE.

## Controls

| Action | Control Keys |
| :--- | :--- |
| **Move Left** | `A` |
| **Move Right** | `D` |
| **Move Up** | `W` |
| **Move Down** | `S` |
| **Fire Laser / Restart** | `Spacebar` |

## Tech Stack & Deployment

- **Engine**: Godot Engine (v4.6)
- **Programming Language**: GDScript
- **Export Target**: HTML5 / WebAssembly (WASM)
- **Deployment**: Automatic deployment to GitHub Pages via GitHub Actions.
